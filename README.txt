# Empire Attacks: Tower Defense

A brower-based tower defense game built with HTML and JavaScript

"In a world where evil remains rampant, the King orders a final attack to reclaim the corrupt land and bring glory to the empire."

## Game Features

- Wave-based gameplay: Survive 4 increasingly difficult rounds
- Tower Defense mechanics: Place defense towers on designated tiles
- Resource management: Earn coins by defeating enemies and spend them on new towers
- Health system: Start the game off with 10 hearts, lose one for each enemy that reaches the end
- Progessive difficulty: Each wave spawns more enemies than the last

## How to Play

1. Start the Game: 
    - Click "START GAME" button on the tittle screen
2. Place Towers:
    - Click on the white highlighted tiles to place towers (each cost 50 coins)
    - Towers automatically target and shoot enemies in range
3. Defend Your Base:
    - Prevent enemies from reachinf the right side of the screen
    - Each enemy that escapes costs you 1 heart
4. Earn Resources:
    - Each defeated enemy awards 19 coins
    - Use coins to build more towers
5. Win Condition:
    - Survive all 4 waves without losing all of your hearts
6. Lose Condition:
    - Reach 0 hearts

## Controls

 - Mouse Movement: Highlight available tower placement tiles
 - Left Click: Place tower on highlighted tile (only if you have enough coins)
